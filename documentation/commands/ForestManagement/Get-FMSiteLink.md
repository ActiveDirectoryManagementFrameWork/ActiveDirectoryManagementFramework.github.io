# Get-FMSiteLink

## SYNOPSIS
Returns the configured link between two sites.

## SYNTAX

```
Get-FMSiteLink [[-SiteName] <String>] [<CommonParameters>]
```

## DESCRIPTION
Returns the configured link between two sites.

## EXAMPLES

### EXAMPLE 1
```
Get-FMSiteLink
```

Returns all configured sitelinks.

## PARAMETERS
