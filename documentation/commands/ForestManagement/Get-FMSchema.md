# Get-FMSchema

## SYNOPSIS
Returns the list of registered Schema Extensions.

## SYNTAX

```
Get-FMSchema [[-Name] <String>] [<CommonParameters>]
```

## DESCRIPTION
Returns the list of registered Schema Extensions.

## EXAMPLES

### EXAMPLE 1
```
Get-FMSchema
```

Returns a list of all schema extensions
