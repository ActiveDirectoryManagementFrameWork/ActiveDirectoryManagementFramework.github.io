# Get-DCShare

## SYNOPSIS
Returns the list of registered shares.

## SYNTAX

```
Get-DCShare [[-Name] <String>] [<CommonParameters>]
```

## DESCRIPTION
Returns the list of registered shares.

## EXAMPLES

### EXAMPLE 1
```
Get-DCShare
```

Returns the list of registered shares.
