# Get-DMPasswordPolicy

## SYNOPSIS
Returns the list of configured Finegrained Password policies defined as the desired state.

## SYNTAX

```
Get-DMPasswordPolicy [[-Name] <String>] [<CommonParameters>]
```

## DESCRIPTION
Returns the list of configured Finegrained Password policies defined as the desired state.

## EXAMPLES

### EXAMPLE 1
```
Get-DMPasswordPolicy
```

Returns all defined PSO objects.
