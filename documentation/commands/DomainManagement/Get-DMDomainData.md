# Get-DMDomainData

## SYNOPSIS
Returns registered domain data gathering scripts.

## SYNTAX

```
Get-DMDomainData [[-Name] <String>] [<CommonParameters>]
```

## DESCRIPTION
Returns registered domain data gathering scripts.

## EXAMPLES

### EXAMPLE 1
```
Get-DomainData
```

Returns all registered domain data gathering scripts
