# Get-DMObject

## SYNOPSIS
Returns configured active directory objects.

## SYNTAX

```
Get-DMObject [[-Path] <String>] [[-Name] <String>] [<CommonParameters>]
```

## DESCRIPTION
Returns configured active directory objects.

## EXAMPLES

### EXAMPLE 1
```

```

Returns all registered objects
