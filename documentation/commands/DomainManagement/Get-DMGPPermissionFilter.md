# Get-DMGPPermissionFilter

## SYNOPSIS
Lists the registered Group Policy permission filters.

## SYNTAX

```
Get-DMGPPermissionFilter [[-Name] <String>] [<CommonParameters>]
```

## DESCRIPTION
Lists the registered Group Policy permission filters.

## EXAMPLES

### EXAMPLE 1
```
Get-DMGPPermissionFilter
```

Lists all registered Group Policy permission filters
