# Get-DMContentMode

## SYNOPSIS
Returns the current domain content mode / content handling policy.

## SYNTAX

```
Get-DMContentMode [<CommonParameters>]
```

## DESCRIPTION
Returns the current domain content mode / content handling policy.
For more details on the content mode and how it behaves, see the description on Set-DMContentMode

## EXAMPLES

### EXAMPLE 1
```
Get-DMContentMode
```

Returns the current domain content mode / content handling policy.
