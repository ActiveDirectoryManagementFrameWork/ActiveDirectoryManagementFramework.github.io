# Clear-DMConfiguration

## SYNOPSIS
Clears the configuration, removing all registered settings.

## SYNTAX

```
Clear-DMConfiguration [<CommonParameters>]
```

## DESCRIPTION
Clears the configuration, removing all registered settings.
Use this to clean up, e.g.
when switching to a new configuration set.

## EXAMPLES

### EXAMPLE 1
```
Clear-DMConfiguration
```

Clears the configuration, removing all registered settings.
